# 제품 요구사항 문서(PRD)

## 1. 제품 한줄 요약
인디해커가 키워드만 입력하면 영어·한국어 블로그 글, SEO 메타데이터, 브랜드 톤까지 5분 내 완성되는 AI 콘텐츠 생성 SaaS.

## 2. 비전 & 목표
- 작은 팀·솔로 창업자가 콘텐츠 마케팅보다 제품 개발에 집중할 수 있도록 글 작성 시간을 90% 이상 단축한다.
- 사용자가 ‘월간 완성 글 수’를 꾸준히 늘려 사업 성장을 체감하도록 한다.

## 3. 핵심 사용자(Persona)
| 구분 | 특징 |
|------|------|
| Indie Hacker Sam | 1인 SaaS 창업자, 연 매출 $1~50K, 블로그는 WordPress 또는 Notion으로 운영, 글쓰기·SEO 경험 부족, 월 $50 이하 툴 사용 예산 |

## 4. 주요 Use Case
1. Sam이 제품 키워드와 간단한 설명을 입력한다.
2. 서비스가 브랜드 보이스 위저드를 통해 톤·길이·독자 수준을 설정한다.
3. 5분 내 초안, 제목, H 태그 구조, 메타디스크립션을 생성한다.
4. Sam은 원스크린 편집기에서 문단별 재생성을 하고 실시간 SEO 체크를 확인한다.
5. 마크다운으로 내보내어 노션 블로그에 붙여넣고 발행한다.
6. 대시보드에서 월간 완성 글 수와 누적 시간 절약을 확인한다.

## 5. 핵심 기능 (MVP)
1. AI 주제 기반 글 자동 생성 (1,500~2,500단어)
2. SEO 키워드 & 메타데이터 자동 삽입
3. 스타일 가이드(브랜드 보이스) 저장·관리
4. 온보딩 위저드(5문항)
5. 원스크린 글 생성·수정 편집기 + 실시간 SEO 점검
6. 시간 절약 현황판 & 월간 완성 글 수 그래프
7. Markdown 내보내기 (외부 API 연동 없음)

## 6. 비MVP / 향후 기능
- 자동 이미지 삽입, WordPress 1-click 발행, 키워드 성과 대시보드, 콘텐츠 캘린더, 팀 협업, 다국어 5개 이상 지원

## 7. 성공 지표(KPI)
- 활성 사용자당 월간 완성 글 수 ≥ 4편
- 글 3편 무료 체험 후 유료 전환율 ≥ 25%
- NPS ≥ 40
- 글 작성 소요 시간 평균 5분 이하

## 8. 가격 정책
- 무료: 글 3편 생성까지, 기능 전체 사용 가능
- 프로 플랜: 월 $19, 글 무제한, 스타일 가이드 3개, 향후 기능 우선 접근

## 9. 기술 스택(기본)
Next.js 15, Hono.js, Supabase(PostgreSQL), TypeScript, TailwindCSS, shadcn, lucide-react, @tanstack/react-query, OpenAI GPT-4o API

## 10. 개발 전략
- 방식: MVP → 사용자 피드백 기반 애자일 개선
- 일정: 8주 내 베타, Product Hunt 런칭
- 팀: 2인(풀스택 1, SEO/PM 1)

## 11. 제약 사항
- 반응형 웹 필수(모바일 ≥ 320px)
- 영어·한국어 고품질 출력 확보
- 일 10만 단어까지 AI API 비용 $200/월 이내 유지

## 12. 위험 & 대응
| 리스크 | 대응 |
|---------|------|
| AI 모델 평준화 | 스타일 가이드 USP 고도화, 편집 UX 최적화 |
| 사용자 이탈 | 시간 절약 현황판·성과 알림으로 지속 가치 제공 |
| 비용 초과 | 단어 수 하드쿼터, Prompt 최적화 |

## 13. 향후 로드맵(12개월)
- M0-2: MVP 개발, 폐쇄 베타 100명
- M3: Product Hunt 공개, 유료 전환 시작
- M4-6: 콘텐츠 캘린더, 워드프레스 발행
- M7-12: 성과 대시보드, 이미지 AI, 팀 협업, 다국어 확대
